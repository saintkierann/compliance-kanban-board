# CONFORM — Backlog board

The CONFORM MVP Kanban board. Built with React + Vite.

The board saves automatically in your browser (storage key `conform-kanban-v3`).
Note: saved tickets live in the browser you used. Open the site in a different
browser or on a different device and you'll see the starter backlog, not your edits.
A shared board across devices/people is a later step (needs a small database).

## Run it on your own computer (optional)
1. Install Node.js (nodejs.org)
2. In this folder: `npm install`
3. `npm run dev` and open the link it prints.

## Going live
Hosted on Vercel, deployed automatically from this GitHub repo.
Build command: `npm run build` — Output directory: `dist`.
